# Statistical Analysis (A/B Test)

This folder contains the statistical analysis for the A/B testing experiment.

## Contents
- statistical_analysis.ipynb: main analysis notebook
- willingness.csv: survey data
- embarrassment.csv: survey data
- Behavior.csv: behavioral data
- survey_results_summary.csv: processed survey results
- behavior_results_summary.csv: processed behavior metrics

## Methods
- t-test
- Mann–Whitney U test
- Effect size (Cohen’s d)
- Confidence intervals
- Proportion z-test
- Funnel analysis
